# ❓ FAQ Accordion

Proyecto de maquetación web basado en el desafío **"FAQ Accordion"** de Frontend Mentor.  
El objetivo fue construir un componente interactivo tipo accordion utilizando HTML, SCSS y JavaScript moderno.

---

## 📌 Descripción

Este proyecto consiste en una sección de preguntas frecuentes (FAQ) donde los usuarios pueden expandir y contraer respuestas de forma dinámica.

Se puso especial énfasis en:
- Estructura semántica  
- Interactividad con JavaScript  
- Diseño responsive  
- Código limpio y escalable  

---

## 🚀 Demo

👉 https://faq-accordion-aldosz94.netlify.app/

---

## 🛠️ Tecnologías utilizadas

- HTML5  
- SCSS (Sass)  
- JavaScript  
- Metodología BEM  
- Vite  

---

## 🎯 Características principales

- 📱 Diseño completamente responsive  
- ❓ Accordion interactivo con JavaScript  
- ➕➖ Cambio dinámico de íconos  
- 🎨 Arquitectura de estilos modular con SCSS  
- 🏷️ Uso de metodología BEM para clases  
- ⚡ Entorno de desarrollo moderno con Vite  

---

## 🧠 Aprendizaje

Durante este proyecto puse en práctica conceptos clave como:

- Manipulación del DOM con JavaScript  
- Uso de `classList.toggle()` para interactividad  
- Cambio dinámico de atributos con JavaScript  
- Organización de estilos con SCSS  
- Aplicación de la metodología BEM  
- Desarrollo de interfaces responsive  
- Uso de herramientas modernas como Vite
